# ⚡ Cybernetic Interface Portfolio

> **A futuristic cyberpunk-themed developer portfolio built with React, Tailwind CSS, Framer Motion, and WebGL-inspired interactive components.**

Designed to deliver an immersive sci-fi experience through animated HUD interfaces, interactive profile scanning, parallax effects, encrypted text animations, and a cinematic hyperspeed background.

# ⚙ Installation

## Clone the Repository

```bash
git clone https://github.com/your-username/cybernetic-interface-portfolio.git
```

```bash
cd cybernetic-interface-portfolio
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

## Production Build

```bash
npm run build
```
